Working with ZIP codes

This dataset contains zip codes. To make working with zip codes easier, we use the zipcodeR package.

library(zipcodeR)

Let’s find the centroid of the La Jolla 92037 (i.e. UC San Diego) ZIP code area.

geocode_zip('92037')
